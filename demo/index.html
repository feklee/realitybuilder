Reality Builder Demo: Instructions

Steps

  1. Start hosting of the Reality Builder widget.
  2. Initialize the database with demo data.
  3. Play with the demo interface:

    Note about the administration controls: The date-time is shown in the time zone of the system/browser.

Supported Browsers

Tested with (in default configurations; versions current as of release of this Reality Builder version):

  • Google Chrome
  • Firefox
  • Opera
  • Internet Explorer 8: see below
  • Internet Explorer 9

Internet Explorer 8 Support

To make the demo work with Internet Explorer 8, FlashCanvas Pro is required. For reasons of license, this library is not included in the source code distribution. If desired, install it as follows:

  1. Download FlashCanvas Pro 1.5 or compatible.
  2. Place FlashCanvas Pro into: ../separate/FlashCanvasPro
