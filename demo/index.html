Supported Browsers

Tested with (in default configurations; versions current as of release of this Reality Builder version):

  • Google Chrome
  • Firefox
  • Opera
  • Internet Explorer 8: see below
  • Internet Explorer 9

Internet Explorer 8 Support

To make the demo work with Internet Explorer 8, the FlashCanvas Pro library is required. For reasons of license, this library is not included in the Reality Builder distribution. If desired, install FlashCanvas Pro as follows:

  1. Download FlashCanvas Pro 1.5 or compatible.
  2. Place FlashCanvas Pro into: ../separate/FlashCanvasPro
